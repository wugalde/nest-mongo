# Inicializar el proyecto pokedex

1. Clonar repo
2. Ejecutar
```
  yarn install
```
3. Instalar nest cli
```
  npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
  docker compose up -d
```

## Stak Usado
* Mongo db
* Nest
* Table Plus
